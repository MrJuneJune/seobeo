Simple to Use

Get a basic HTTP server up and running in just a few lines of code

/* A minimal Cerver example */ #include <cerver/cerver.h> void handle_request(request_t* req, response_t* res) { response_set_status(res, 200); response_set_header(res, "Content-Type", "text/plain"); response_set_body(res, "Hello from Cerver!"); } int main() { /* Initialize the server */ cerver_t* server = cerver_create(); /* Configure the server */ cerver_set_port(server, 8080); cerver_set_connection_limit(server, 1000); /* Add a route */ cerver_add_route(server, "GET", "/", handle_request); /* Start the server */ cerver_start(server); /* Wait for server to complete */ cerver_wait(server); /* Clean up */ cerver_destroy(server); return 0; }

Get Started

Follow these simple steps to get Cerver up and running

1

Install Dependencies

Make sure you have a C compiler and build tools installed on your system.

sudo apt-get install build-essential cmake
2

Clone the Repository

Get the source code from GitHub.

git clone https://github.com/your-username/cerver.git
3

Build the Library

Compile the source code.

cd cerver && mkdir build && cd build && cmake .. && make
4

Install the Library

Install the library on your system.

sudo make install
5

Create Your First Server

Copy the example above and compile it.

gcc -o myserver myserver.c -lcerver
6

Run Your Server

Start your HTTP server.

./myserver
